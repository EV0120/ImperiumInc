# Imperium Inc. - Tienda en Línea

Este repositorio contiene mi trabajo como desarrollador front-end en Imperium Inc., mostrando el código fuente y la documentación del sitio web de la tienda en línea. Aquí encontrarás todos los recursos necesarios para la instalación, uso y contribuciones al proyecto.

### Acerca de Mí
Soy un desarrollador front-end con experiencia en la creación de interfaces de usuario atractivas y funcionales. En mi trabajo con Imperium Inc., he aplicado mis habilidades para desarrollar un sitio web eficiente y fácil de usar.

### Habilidades
- **Desarrollador Front-End**:
  - **Tecnologías**: HTML, CSS, JavaScript , Angular.
  - **Herramientas**: Visual Studio Code, GitHub, Google Tag Manager
  - **Experiencia**: Desarrollo de sitios web, optimización de rendimiento y accesibilidad

### Proyectos Destacados
- **Sitio Web de la Tienda en Línea Imperium Inc.**
  - **Descripción**: Desarrollo de un sitio web para la tienda en línea de Imperium Inc. Incluye catálogo de productos, sistema de contacto y enlaces a redes sociales.
  - **Idiomas Disponibles**: Español.
  - **Contenido del Sitio**:
    - **Sección de Catálogo**: Descripción y precios de los productos.
    - **Sistema de Contacto**: Enlaces a WhatsApp, Facebook, Instagram y TikTok.
    - **Políticas de Privacidad**: Información sobre términos y condiciones y políticas de privacidad.
